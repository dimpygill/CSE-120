1. Artificial Intelligence Artificial Intelligence (AI) is the field of computer science that focuses on building systems that can perform tasks that normally require human intelligence. These tasks include: • Understanding language • Recognizing images • Making decisions • Solving problems • Learning from experience AI does NOT “think” like humans. It: • finds patterns • uses data • makes statistical decisions 2. Major Categories of AI AI is a big umbrella, and it is divided into several major categories. CATEGORY 1: Machine Learning (ML) Machine Learning is a branch of AI where machines learn patterns from data instead of being explicitly programmed. Instead of writing rules, we give data and let the machine learn rules itself. Example: • Email spam detection • Credit card fraud detection • Movie recommendations Types of Machine Learning: 1. Supervised Learning Model learns from labeled data. For example: • Input: Email • Output: Spam / Not Spam Tasks: • Classification • Regression 2. Unsupervised Learning Model finds hidden patterns without labels. For example: • Customer segmentation • Grouping similar users Tasks: • Clustering • Association rules 3. Reinforcement Learning Model learns by reward and punishment Example: • Game AI (chess, AlphaGo) • Robotics navigation CATEGORY 2: Deep Learning Deep Learning is a subset of machine learning that uses neural networks with many layers. It learns patterns in a layered way (like human brain-like structure). Applications: • Face recognition • Speech recognition • Self-driving cars • ChatGPT-like models Important point: Deep learning works best with: • Large data • High computing power CATEGORY 3: Natural Language Processing (NLP) Natural Language Processing helps machines understand human language. It teaches computers to read, understand, and generate human language. Examples: • Chatbots • Google Translate • Sentiment analysis • Spam detection Subtasks: • Text classification • Language translation • Question answering • Summarization CATEGORY 4: Computer Vision Computer Vision helps machines see and understand images/videos. Examples: • Face recognition • Self-driving cars detecting objects • Medical scan analysis CATEGORY 5: Robotics Robotics combine physical machines that can act in the real world. Key idea behind this is AI + sensors + movement = real-world action Examples: • Industrial robots • Delivery robots • Surgical robots CATEGORY 6: Expert Systems (Rule-Based AI) Older form of AI that uses fixed rules instead of learning. Example: IF temperature > 100 THEN alert fever Uses: • Medical diagnosis systems (early versions) • Decision support systems Limitation: • Cannot learn • Cannot adapt easily CATEGORY 7: Generative AI Generative AI creates new content. Instead of analyzing data, it creates new data. Examples: • Text generation (ChatGPT) • Image generation (DALL·E, Midjourney) • Music generation CATEGORY 8: Agentic AI Agentic AI can: • Set goals • Plan steps • Use tools • Take actions Example: Instead of answering: “Plan a 5-day trip to New York under $1500” It can: • Analyze budget, dates, and preferences • Search for flights using travel websites/APIs • Compare hotel options based on price and location • Create a day-by-day itinerary • Adjust plan if total cost exceeds budget • Check weather and update activities • Provide final itinerary with booking options
